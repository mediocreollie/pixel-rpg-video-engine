# Character Assets

Store future character sprites and portraits here.

Planned assets:

- Still portraits
- Walking sprite sheets
- Optional emote portraits
- Character-specific colour references

Photos of real people may be used as private inspiration, but final assets should be stylised DS-era pixel characters. Keep people recognisable through silhouette, outfit colour, hair shape, and accessories rather than realism.

If reference photos are private or sensitive, store them outside the public repo.

Example names:

- `friend-jack-walk.png`
- `friend-jack-portrait.png`
- `beth-walk.png`
- `beth-portrait-happy.png`
