# Assets

This folder is reserved for future game assets.

Current status: the game uses generated placeholders, not final art, music, or sound.

Planned folders:

- `characters/`: sprites, portraits, walking sheets, emote portraits
- `locations/`: reusable location art and tiles
- `music/`: loopable background music
- `sfx/`: short sound effects
- `ui/`: menu, dialogue, title card, and transition polish assets

Do not commit copyrighted assets unless they are licensed for this project.

Use lowercase kebab-case filenames, such as `friend-jack-walk.png` or `door-enter.wav`.
